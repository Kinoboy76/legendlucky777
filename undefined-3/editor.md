# 진행 순서

{% stepper %}
{% step %}
### <mark style="color:yellow;">서비스최초접속</mark>

* 메시지 시스템을 통한 튜토리얼 입장 링크 수령
* 추후 접속 시 팝업 메시지로 표시 예정
{% endstep %}

{% step %}
#### <mark style="color:red;">무료 포인트 지급</mark>

* 모드 진입시에 무료 포인트 10만 지급
* 포인트 수량은 사용자 피드백과 데이터 분석을 바탕으로 추후 협의 필요
* 모드 시작 전, 무료 포인트의 현금 인출 불가 및 보유 포인트 연동 불가 고지
{% endstep %}

{% step %}
#### <mark style="color:orange;">튜토리얼 진행</mark>

* 게임 조작, 게임 특징 학습
* 게임 주요 피처 체험을 통해 게임에 대한 흥미 유발
{% endstep %}

{% step %}
### <mark style="color:purple;">조건 달성</mark>

* 게임 머니 소진, 총 스핀 수, 진입단계, 특정 피처 완료 등의 조건 완료에 따라 차등 보상 지급
{% endstep %}

{% step %}
### <mark style="color:blue;">보상 지급 및 게임 재 입장 유도</mark>

* 메시지 시스템을 이용한 보상 지급
* 게임 포인트, 프리 스핀을 보상으로 지급
* 해당 게임의 프리 스핀 아이템을 이용하여 게임에 재 입장 유도
{% endstep %}
{% endstepper %}
